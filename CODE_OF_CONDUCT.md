# Code of Conduct

## Our Standard

This project expects respectful, focused, and constructive collaboration.

Examples of positive behavior:

- using welcoming and precise language;
- respecting different experience levels;
- focusing feedback on the work;
- being clear about risks and tradeoffs;
- prioritizing safety for robotics-related changes.

Examples of unacceptable behavior:

- harassment or discrimination;
- personal attacks;
- publishing private information;
- deliberately unsafe guidance;
- disruptive conduct in issues or pull requests.

## Enforcement

Maintainers may remove comments, close issues, or restrict participation when behavior harms the project or its contributors.

## Scope

This code of conduct applies to project repositories, issues, pull requests, discussions, and any official project spaces.
