# v0.2.0 - Cognitive Robotics Foundation

`v0.2.0` prepares CHEVEL AI for a stronger public release without overstating
hardware readiness.

## Highlights

- Adds public version positioning for the cognitive robotics foundation release.
- Makes `data/configs`, `data/models`, and `data/workflows` meaningful with
  example manifests and local-private config guidance.
- Documents the universal CHEVEL scope for Dum-E/U and WIL-E.
- Keeps the README honest: implemented runtime, simulated contracts,
  hardware-ready foundations, and planned adapters are separated clearly.
- Adds runtime config loading from optional local JSON files while preserving
  environment variable overrides.

## Current Runtime

- Local chat and API runtime.
- Ollama-backed LLM engine.
- SQLite and JSON memory.
- Cognitive core modules.
- Safe simulated Dum-E/U bridge.
- 5DOF Arduino arm firmware and Python controller.
- Voice foundations.
- Optional native C++ helper core with Python fallback.

## Safety Note

Real hardware adapters remain opt-in future work. Physical movement must remain
behind confirmation and emergency-stop paths.
